1、计算机相关概念

计算机—能够根据一组指令操作数据的机器

特点：1、功能性—可以进行数据计算；2、可编程性—根据一系列指令来执行

TOP500，www.top500.org—世界超级计算机排行榜，每年两次选出世界上最快的500台计算机。

2、程序设计语言基本概述

程序设计语言，即编程语言，是计算机能够理解和识别操作的一种交互体系。

主要类型：

1、机器语言：01代码，CPU认识的语言，例：2+3即为1101001000111011
2、汇编语言：在机器语言上增加了人类可读的助记符，例：2+3即为add 2,3,result
3、高级语言：向自然语言靠近的语言，例：2+3即可写为result = 2 + 3

4、Python语言的开发环境配置

安装：Python主页—http://www.python.org/download/—下载对应版本的Python。
    （安装注意：安装过程中点选Add python to Path，即将Python配置在环境变量中；否则就需要手动配置Python环境变量）
    
启动：方法一：启动Windows命令行，输入Python即出现对应Python版本信息等，随后在提示符“>>>”后输入语句即可
   　方法二：启动Python的IDLE打开Python图形化界面,可直接在提示符后输入语句(该操作会执行每一条代码)；也可以打开编辑窗口，输入语句并保存，然后进行运行
   　方法三：按照Python语法格式编写代码，可采用任意文本编辑器，然后将其保存为“.py”文件
   　方法四：使用集成开发工具，如：PyCharm
